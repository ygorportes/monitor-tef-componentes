# Monitor TEF com Componentes

Aplicação simples desenvolvida em React para a atividade prática.

## Sobre o projeto

Este projeto simula uma tela de monitoramento dos principais serviços de um ambiente TEF.

A aplicação foi criada com o objetivo de praticar a criação e utilização de componentes em React.

## Componentes criados

- App.jsx: componente principal da aplicação
- StatusCard.jsx: componente secundário utilizado para exibir o nome e o status de cada serviço

## Tecnologias utilizadas

- React
- JavaScript
- CSS
- Vite
